反编译app

既然我们想想研究一下哔咔，看那简介的官网界面已经生无可恋了，那么只能从它提供的Android客户端来入手啦。

工具介绍

apktool
用作资源文件获取，可以提取出图片文件和布局文件进行使用查看

dex2jar
将apk反编译成java源码(classes.dex转化成jar文件)

jd-gui
查看APK中classes.dex转化成出的jar文件，即源码文件
在这里，我主要用到的是dex2jar和jd-gui这两个工具

反编译成Java源码

  • 解压.apk文件得到classes.dex;
  • 用dex2jar的一条命令d2j-dex2jar.sh ~/classes.dex ~/轻松得到Java源码
